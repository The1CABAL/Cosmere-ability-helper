Tip (Combat): you can use one triggered action per round when its trigger occurs; free triggered actions do not count toward that limit. Mark “Uses non-free triggered” for abilities that share the one-per-round non-free triggered budget.

Scratch pad

Type like a document, or switch to Ink and draw with finger or stylus (pressure adjusts line weight). Use Erase mode or the stylus eraser / barrel button to rub out ink. Pages are saved automatically—pick a day’s page or add another for the same session.

Rules search

Search Markdown from SteelCompendium/data-md (community Draw Steel rules data; not affiliated with MCDM). Use GitHub online, or link a local git clone of that repo to search file names and full text on disk (no GitHub code-search rate limits). Pick a result to load the document.
